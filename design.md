## 二、核心功能需求

### 2.1 轉盤系統
- 12 等分扇形轉盤
- 每格顯示：emoji + 獎品名稱（簡短）
- 中心顯示：「2026.04.03 ♡」
- 轉針位於正上方，可點擊觸發

### 2.2 轉動機制
| 項目 | 規格 |
|:---|:---|
| 觸發方式 | 點擊轉針 |
| 旋轉圈數 | 5-7 圈隨機 |
| 減速曲線 | ease-out（先快後慢）|
| 總時長 | 5-7 秒 |
| 停止邏輯 | 隨機落在任一格，機率均等 |

### 2.3 中獎彈窗
- 從中心放大彈出（scale 0→1 + opacity）
- 顯示：大 emoji、獎品名稱、詳細說明
- 「再轉一次」按鈕（冷卻 1 秒防連點）
- 特殊音效（可選）

## 四、彩蛋系統：「弄壞針針」

### 4.1 觸發條件
- 連續快速點擊轉針 7 次以上
- 每次點擊間隔 < 300ms

### 4.2 階段效果

| 階段 | 點擊次數 | 視覺效果 | 文字回饋 |
|:---|:---|:---|:---|
| 警告 | 3 | 轉針微微晃動 | 「好像有點鬆了...？」 |
| 危險 | 5 | 轉針歪斜、火花特效 | 「喂！要壞掉了啦！」 |
| 崩潰 | 7 | 針飛出去（旋轉飛離畫面）| 「啊啊啊——」 |
| 結局 | - | 轉盤自動狂轉 5 圈後全亮 | 「既然針壞了... 那這些全都給你吧 ❤️」 |

### 4.3 壞掉後狀態
- 所有格子同時發光脈衝
- 畫面飄下愛心 confetti
- 顯示「大獎解鎖」清單：

  - 無限色色券（延期至週末）

### 4.4 恢復機制
- 點擊「修好針針」按鈕重置
- 或重新整理頁面

### 5.1 配色系統

| 用途 | 色碼 | 說明 |
|:---|:---|:---|
| 背景 | `#FFF5F5` | 奶油白 |

| 輔色 | `#F4E4C1` | 香檳金 |
| 強調 | `#C45C5C` | 深玫瑰 |
| 特殊格 | `#DC143C` | 深紅（無限色色券）|
| 文字 | `#4A4A4A` | 深灰 |

### 5.2 字體
- 中文：系統預設（蘋方、微軟正黑體）
- 標題可選：Google Fonts「Noto Sans TC」

### 5.3 轉盤尺寸
- 手機：寬度 90vw，最大 350px
- 桌面：固定 400px
- 轉針：置頂，三角形，大小 40px

### 5.4 動畫規格

| 效果 | 參數 |
|:---|:---|
| 轉盤旋轉 | `transform: rotate(deg)`，使用 `cubic-bezier(0.25, 0.1, 0.25, 1)`|
| 彈窗出現 | `transform: scale(0) → scale(1)`，300ms |
| 格子發光 | `box-shadow` 脈衝動畫，1.5s 循環 |
| Confetti | Canvas 粒子系統或 CSS 動畫 |

## 七、技術實作建議

### 7.1 檔案結構
```
single-file.html
├── <style> CSS（全部內嵌）
├── <body>
│   ├── 開場畫面 div
│   ├── 轉盤容器 div
│   │   ├── SVG/Canvas 轉盤
│   │   └── 轉針 div
│   ├── 中獎彈窗 div（預設 hidden）
│   └── Confetti canvas（預設 hidden）
└── <script> JavaScript（全部內嵌）
```

### 7.2 核心演算法

**轉盤旋轉角度計算**
```javascript
// 每格 30 度（360/12）
// 目標格子的中線角度 = (index * 30) + 15
// 加上隨機圈數（5-7 圈 = 1080-1800 度）
// 最終角度 = 隨機圈數 + (360 - 目標格子中線角度) + 當前角度
```

**中獎判定**
```javascript
// 旋轉結束後，轉針指向正上方（0度/360度）
// 計算實際旋轉後的角度 % 360
// 除以 30 取整數，得到獎品 index
```

### 7.3 響應式斷點
| 裝置 | 寬度 | 調整 |
|:---|:---|:---|
| 手機 | < 480px | 轉盤 90vw，字體 14px |
| 平板 | 480-768px | 轉盤 350px，字體 16px |
| 桌面 | > 768px | 轉盤 400px，字體 18px |
